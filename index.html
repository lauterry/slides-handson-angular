    </body>
</html>

... et vérifier que ca marche


<html ng-app>

</html>

Data Binding bi-directionnel

    <div ng-controller="myController">
        <input type="text" ng-model="greeting">
        <p>{{greeting}}</p>
    </div>
</html>
                        

function myController($scope){
    $scope.greeting = 'Hello World';
}

  • Une application contient un seul scope global appelé $rootScope
  • mais plusieurs $scope enfants
  • Un scope est créé pour chaque contrôleur
  • Un scope créé est toujours enfant d'un scope parent ...
